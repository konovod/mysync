# mysync

UDP protocol library for games, inspired by http://gafferongames.com articles (but missing all things i don't need in my games). Totally WIP.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  mysync:
    github: konovod/mysync
```
